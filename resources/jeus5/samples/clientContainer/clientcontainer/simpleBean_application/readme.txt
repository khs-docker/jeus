I. 예제 실행 방법
 
   * Build 준비
    >jant createtable

   * Build 및 Deploy, 클라이언트 실행
    > jant

   * 클라이언트만 실행
    > jant run

   * Undeploy
     > jant undeploy

   * 생성된 디렉토리 및 파일 삭제
     > jant clean

II. 예제 설명

   * simpleBean(Entity bean)을 호출하는 client application 이다.
      bean을 호출하고 결과를 화면에 보여준다.
